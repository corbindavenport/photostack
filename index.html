Welcome to PhotoStack!

Welcome to PhotoStack, the free and open-source batch photo editor.

To get started, click the Import button at the bottomtop-right to import some images. When you're ready to save your work, click the Export button.

About

PhotoStack

PhotoStack is developed by Corbin Davenport, and all the code is freely available as open-source. If you want to report a bug or suggest features, please create an issue on GitHub or join the Discord group.

Keyboard shortcuts

PhotoStack offers keyboard shortcuts for common functions. These may not work with all browserson all devices. Also, they will not work if any popups (like the watermark editor or this help menu) are open.

Command Shortcut
Import local images Shift + O
Clear imported images Shift + X
Open watermarks Shift + W

Important info for Safari

Safari 13.1+ on macOS, as well as Safari on iOS/iPadOS 13.4+, introduced a seven-day cap on all locally-stored data by websites. This means that unless you open PhotoStack at least once a week, your saved watermarks could be deleted.

The only way to avoid losing your watermarks is to export them as files (from the watermark list), then re-import them after Safari wipes PhotoStack's data.

In addition, PhotoStack may not be able to download files (completed images, exported watermarks, etc.) when it is opened from the iOS home screen. To get around this, open edit.photostack.app in your browser instead.
